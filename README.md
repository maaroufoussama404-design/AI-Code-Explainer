# AI Code Explainer

AI Code Explainer is a web application built with Python and Flask that helps developers better understand and improve their code.

Users can paste source code, select a programming language, and receive an automated analysis that includes:

* Code explanation
* Common bug detection
* Code quality score
* Improvement suggestions

The project demonstrates backend development, web application design, code analysis logic, and user interface development.

## Features

* Multi-language support (Python, JavaScript, Java, C++)
* Code quality scoring system
* Common issue detection
* Improvement recommendations
* Simple and intuitive user interface

## Technologies Used

* Python
* Flask
* HTML
* CSS
* Git & GitHub
